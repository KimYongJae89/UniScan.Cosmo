git_bundle_create.bat
현재 리포지토리의 master 브랜치 번들을 만든다. (yyyyMMdd_HHmmss.bundle)
브랜치 히스토리를 모두 저장. (용량 큼)

git_bundle_create_diff.bat
현재 리포지토리의 master 브랜치 번들을 만든다. (yyyyMMdd_HHmmss.diff.bundle)
origin/master 브랜치(온라인 git과 동기화 했을 때의 master 브랜치 상태)와 master 브랜치의 차이만 저장. (용량 작음)


git_pull.bat
번들을 현재 리포지토리의 masterm 브랜치로 합친다.
더블클릭 실행: 같은 폴더의 unieye.bundle 파일 사용.
번들 파일을 bat 파일 위로 드래그-드롭: 해당 번들 파일을 사용.
